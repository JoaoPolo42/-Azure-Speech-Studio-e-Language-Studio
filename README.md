# Azure Speech & NLP Lab
Repositório de estudos práticos em processamento de fala e linguagem natural usando Azure Speech Studio e Language Studio

# Visão Geral
Este repositório contém anotações, exemplos de código e tutoriais para desenvolver soluções de IA usando:
✔ Azure Speech Studio (reconhecimento de fala, síntese de voz, tradução)
✔ Azure Language Studio (análise de sentimentos, extração de entidades, chatbots)

O objetivo é documentar aprendizados e fornecer um guia estruturado para implementações futuras.

# Funcionalidades Exploradas
## Azure Speech Studio
Speech-to-Text (STT): Transcrição de áudio em tempo real e em batch.

Text-to-Speech (TTS): Geração de vozes naturais e personalizadas.

Tradução de Fala: Conversão entre idiomas em tempo real.

Custom Speech: Adaptação de modelos para domínios específicos.

# Azure Language Studio
Análise de Sentimentos: Classificação de polaridade (positivo/neutro/negativo).

Reconhecimento de Entidades (NER): Identificação de pessoas, locais, datas.

Resumo Automático: Extração de pontos-chave de textos longos.

Chatbots Inteligentes: Construção de assistentes com QnA Maker e CLU.
